What is ASRR project?

ASRR project aims to provide Miscellaneous R functions for Applied Sociological Research. It stands for Applied Sociological Research with R.

Since 28 Sep 2009, QCA3 is forked from this project and uploaded to CRAN. QCA3 contains a set of functions is for Qulitative Comparative Analysis (QCA). It can used for all three types of QCA, namely csQCA, mvQCA and fsQCA. Examples from Rihoux and Ragin (2009) can be reproduced by QCA3. Download

How to cite the software

use citation('pkgname') to get the citation information, for example, citation('QCA3').
