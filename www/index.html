What is ASRR?

ASRR is a R package. It aims to provide Miscellaneous R functions for Applied Sociological Research. It stands for Applied Sociological Research with R. The most important set of functions is for Qulitative Comparative Analysis (QCA). It can used for all three types of QCA, namely csQCA, mvQCA and fsQCA.You can find examples of QCA here. ASRR is released as free software under the New style BSD license.

Installation

Launch R, then use R command to install ASRR: install.packages("ASRR",repos=" http://R-Forge.R-project.org",dependencies=TRUE) .

How to cite the software

Ronggui HUANG (2009). ASRR: Applied Sociological Research with R. R package version 0.0-1. http://asrr.r-forge.r-project.org/
